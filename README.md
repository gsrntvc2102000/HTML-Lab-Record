# HTML-Lab-Record
Web Technology Lab


1. **Web Programming Syllabus:**

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Web Programming Syllabus</title>
</head>
<body bgcolor="Grey">
    <h1 align="center">Web Programming Syllabus</h1>

    <h2>Fundamentals of Web Programming</h2>
    <p><b><u>Unit 1:</u></b> Internet, World Wide Web (WWW), Web Browsers, and Web Servers</p>
    <p><b>Topics:</b> Understanding the Internet, Introduction to WWW, Web Browsers, Web Servers</p>

    <p><b><u>Unit 2:</u></b> Introduction to HTML and XHTML</p>
    <p><b>Topics:</b> HTML and XHTML Basics, Hyperlinks, Forms, Frames, Differences between HTML and XHTML</p>

    <p><b><u>Unit 3:</u></b> Cascading Style Sheets (CSS)</p>
    <p><b>Topics:</b> Syntax of CSS, Styling Web Pages, Text Formatting, Conflict Resolution</p>

    <h2>JavaScript Fundamentals</h2>
    <p><b><u>Unit 4:</u></b> Overview of JavaScript</p>
    <p><b>Topics:</b> Object-Oriented JavaScript, Syntax, Screen Output, Keyboard Input, Objects, Arrays, Functions</p>

    <p><b><u>Unit 5:</u></b> JavaScript and HTML Integration</h2>
    <p><b>Topics:</b> Creating a Web Page to Display the Syllabus</p>
</body>
</html>
```
